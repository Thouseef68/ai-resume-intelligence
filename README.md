🚀 AI Resume Intelligence

An AI-powered resume analysis platform that evaluates resumes, provides actionable feedback, and enhances job readiness using advanced LLMs.

⸻

🔍 Features
	•	📄 Resume Parsing & Analysis
	•	🤖 AI Feedback using LLMs (Gemini / OpenRouter / Groq)
	•	📊 Skill Gap Detection
	•	🎯 Job Match Insights
	•	💡 Improvement Suggestions

⸻

🛠️ Tech Stack
	•	Python
	•	Streamlit
  •	ML(TF-IDF,LogisticRegression,EmbeddingMatching,XGBoost)
  •	Scikit-learn
	•	Pandas & NumPy
	•	LLM APIs (Gemini, OpenRouter, Groq)
